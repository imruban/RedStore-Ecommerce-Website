RedStore E-commerce Website

Online Shopping is a likestyle e-commerce web application, which retails various fashion and lifestyle products (Currently Men's Wear). This project allows viewing various products available enables registered users to purchase desired products instantly using Paypal payment processor (instant pay) and also can place order by using Cash on Delivery (Pay Later) option. This project provides an easy access to Administrators and Managers to view orders placed using Pay Later and instant Pay options. In this project I use C#.Net, SQL Server 2008 and various components of ASP.NET.
